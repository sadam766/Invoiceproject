# **App Name**: Sales Dashboard

## Core Features:

- Dashboard Overview: Display key performance indicators (KPIs) such as total revenue, total customers, total products, and conversion rate.
- Sales Overview Chart: Visualize sales data over time with a line chart, showing revenue and sales trends.
- Top Products: List the top-selling products with sales figures and quantities.
- Recent Sales: Show a table of recent sales transactions with details like invoice ID, customer, date, amount, and status.
- Data Filtering: Allow users to filter and view sales data by different time periods.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) for a calm and professional feel, similar to the provided image.
- Background color: Off-white (#FAFAFA) to maintain a clean and readable interface.
- Accent color: Light Blue (#ADD8E6) for highlighting interactive elements and important data.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable design.
- Use minimalist, outline-style icons to represent different data categories and actions.
- Employ a card-based layout to organize and present different data sections in a structured manner.
- Use subtle transitions and animations for loading data and interactions to improve user experience.
